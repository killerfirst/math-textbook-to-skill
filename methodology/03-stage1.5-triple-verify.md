# 阶段 1.5 — 三重验证（数学教材版）

## 目标

数学教材的验证主要价值**从「淘汰」变为「去重 + 标注依赖重要性」**。因为定理和例题是客观存在的，不是"作者的见解"，所以通过率天然高（85-100%）。

## 三重验证（数学适配版）

### V1 — 跨章引用（Cross-chapter Citation）

**问题**：这个定理在书中**至少被另一章的定理引用过**吗？

- **通过**：维数公式在第 5 章（特征值）、第 7 章（谱定理）中被引用 → 通过
- **不通过**：某个引理只在定义它的那一章用了一次 → 降级嵌入父定理 skill
- **例题**：体现的方法是否有至少 2 个变体题目？（不是仅数字不同）

### V2 — 预测力（Predictive Power）

**问题**：这个定理/方法能否解决同类型的变体题目？

- 自己设计一个书中没直接讨论过的变体题目
- **通过**：能推导出正确答案 → 通过
- **不通过**：解法只在原文那一道题上有效 → 降级

### V3 — 独特性（Exclusivity）

**问题**：低年级教材中，是不是高中已知内容？高年级教材中，是不是本领域的基础公理？

- **低年级（大一/大二）**："不是高中课本里学过的"——例如"复数乘法交换律"是高中内容 → 不通过
- **高年级（大三+）**："不是本学科的基础约定"——例如"笛卡尔积"属于集合论约定 → 不通过
- **通过**：线性映射基本定理（维数公式）——线性代数自身的核心结果

## 数学教材的典型淘汰原因

| 淘汰原因 | 示例 |
|---------|------|
| V1 失败 | 例题与其他例题重复度太高（仅数字不同） |
| V2 失败 | 过于冷僻的术语，不会在后续内容中使用 |
| V3 失败 | "复数乘法交换律的验证"——这是复数定义的自然推论 |
| V3 失败 | 纯粹集合论术语（如"笛卡尔积"）——不属于本教材 |
| 去重 | 同一个反例在多个形式中出现（保留最具代表性的） |

## 通过率预期

| 书籍类型 | 典型通过率 | 原因 |
|---------|-----------|------|
| 方法论/商业书 | 25-50% | 大量候选是松散的"建议" |
| 数学教材 | 85-100% | 定理和例题是客观存在的 |

## 常见失败模式

1. **跳过 V1** — 以为"定理肯定是对的就不用验证"（标准跑偏，V1 判断的是重要性而非正确性）
2. **V3 过松** — 只要是书里的定理就通过（应该淘汰属于"预备知识"的内容）
3. **去重遗漏** — 同一概念在反例和边界情形中重复出现只保留一条
